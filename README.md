# 🚀 Mini CRM System (Client Lead Management)

A full-stack CRM (Customer Relationship Management) system built using the MERN stack to manage and track client leads efficiently.

---

## 🌐 Live Demo

- 🔗 Frontend: https://famous-daffodil-ed719b.netlify.app/
- 🔗 Backend: https://future-fs-02-h83h.onrender.com

---

## 🎯 Features

- ✅ Add new leads (name, email, source)
- 🔄 Update lead status (New → Contacted → Converted)
- 📝 Add follow-up notes for each lead
- 🔍 Search leads by name
- 🎯 Filter leads by status
- 📊 Dashboard with:
  - Total leads
  - Contacted leads
  - Converted leads
  - Conversion rate
- ❌ Delete leads
